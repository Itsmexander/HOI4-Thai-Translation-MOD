### FEATURES - คุณลักษณะเด่น
สามารถพิมพ์ไทยได้ ส่วนที่สามารถแก้ไขและบันทึกเป็นภาษาไทย ได้แก่ ชื่อกองพล, ชื่อกองทัพ, ชื่อหมู่กองทัพ, ยุทธบริเวณ, ชื่อเรือรบ, ภารกิจกองเรือ, แม่แบบกองพล ,ชื่อรุ่นโมเดลอาวุธ และ อื่นๆ
